Branch Point initialization

The only way to start a continuation of branch points supported in the current version is to start it from a Branch point detected on an equilibrium curve or on a fold curve. This can be done using the following statement: [x0,v0]=init_BP_BP(@odefile, x, p, ap, bp). This routine stores its information in a global stucture bpds. The result of init_BP_BP contains a vector x0 with the state variables and the three active parameters and a vector v0 that is empty. Here odefile is the ode-file to be used, x is a vector of state variables containing the values of the state variables returned by a previous equilibrium curve or fold curve continuation, p is the vector containing the current values of the parameters, ap is the vector containing the indices of the active parameters and bp is the index of the branch parameter.
