The generated curve is plotted in Figure 32 with the command: 

>> cpl(x,v,s)
In this case x has dimension 2, so a 2D-plot is drawn with the first component of x (value of the state variable ) on the x-axis and the second component of x (value of the state variable ) on the y-axis. The above commands are also executed by running testdrawcurve; the file testdrawcurve.m is in the directory Testruns.
