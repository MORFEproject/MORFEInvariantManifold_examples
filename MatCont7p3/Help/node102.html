Branch Point of Cycles initialization

The only way to start a continuation of branch points supported in the current version is to start it from a BPC detected on an limitcycle curve or on an LPC curve. This can be done using the following statement: [x0,v0]=init_BPC_BPC(@odefile, x, s, ap, ntst, ncol, bp). This routine stores its information in a global stucture lds. The result of init_BP_BP contains a vector x0 with the state variables and the three active parameters and a vector v0 that is empty. Here odefile is the ode-file to be used, x is a vector of state variables containing the values of the state variables returned by a previous limit cycle curve or fold of cycles curve continuation, s is a structure containing the BPC point values returned by a previous limit cycle curve or fold of cycles curve continuation, ap is the vector containing the indices of the active parameters and bp is the index of the branch parameter. ntst and ncol are again the number of mesh and collocation points for the discretization.
