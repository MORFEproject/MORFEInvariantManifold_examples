Time integration

MATCONT uses the MATLAB representation of ODEs, which can also be used by the MATLAB ODE solvers. The user of CL/SMALL>_MATCONT is therefore also able to use his/her model within the standard ODE solver without rewriting the code. In MATCONT, also backward time integration is possible using the standard ODE solvers. To make them accessible in MATCONT, some output functions and properties were created.
